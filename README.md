LC Romanization Converter (Myanmar)
မြန်မာအမည်များနှင့် စာအမည်များကို Library of Congress (LC) Romanization (Transliteration) စံနှုန်းအတိုင်း အလွယ်တကူ ပြောင်းလဲပေးနိုင်သော Tool ဖြစ်ပါသည်။ စာကြည့်တိုက်ပညာရှင်များ၊ PhD သုတေသီများနှင့် Cataloger များအတွက် ရည်ရွယ်၍ ဆရာဇော် (Zaw Aung Htut) မှ ဖန်တီးထားပါသည်။
အဓိက လုပ်ဆောင်ချက်များ (Features)
* Author Mode: စာရေးသူအမည်များကို ရိုက်ထည့်ပါက LC စံနှုန်းအရ အမည်စာလုံးတိုင်း၏ အစကို အကြီးစာလုံး (Title Case) ဖြင့် ပြောင်းလဲပေးပါသည်။ (ဥပမာ - Ūʺ Phe Moṅʻ Taṅʻ)
* Title Mode: စာအမည်များကို ရိုက်ထည့်ပါက ရှေ့ဆုံးစာလုံးတစ်ခုတည်းကိုသာ အကြီး (Sentence Case) ထားရှိပြီး ကျန်စာလုံးများကို LC Transliteration စည်းမျဉ်းအတိုင်း ဖော်ပြပေးပါသည်။ (ဥပမာ - Mranʻmā cāpe samuiṅʻʺ)
* 3rd Indention Support: စာရင်းသွင်းရာတွင် အဆင်ပြေစေရန် စာကြောင်းအစ၌ Indention ပါဝင်ပြီးသား ရလဒ်ကို ထုတ်ပေးပါသည်။
* Quick Actions: ရရှိလာသော Romanization ရလဒ်များကို တစ်ချက်နှိပ်ရုံဖြင့် Copy ကူးယူနိုင်သလို၊ လိုအပ်ပါက Edit ခလုတ်ဖြင့် အလွယ်တကူ ပြန်လည်ပြင်ဆင်နိုင်ပါသည်။
နည်းပညာဆိုင်ရာ အချက်အလက်များ (Technical Stack)
ဤ Tool ကို အောက်ပါ Language များဖြင့် တစ်ဖိုင်တည်း (Single File) စုစည်းတည်ဆောက်ထားပါသည်-
* HTML5: Structure နှင့် Input fields များအတွက်။
* CSS3: User Interface နှင့် Indention formatting များအတွက်။
* JavaScript (Vanilla JS): LC mapping Logic နှင့် Transliteration algorithm များအတွက်။
အသုံးပြုနည်း (How to Use)
* LC_Romanization_Converter.html ဖိုင်ကို Browser (Chrome, Firefox သို့မဟုတ် Edge) တစ်ခုခုဖြင့် ဖွင့်ပါ။
* "စာရေးသူအမည်" သို့မဟုတ် "စာအမည်" ကွက်တွင် မြန်မာလို ရိုက်ထည့်ပါ။
* အောက်ပါ Result Box တွင် LC စံနှုန်းအတိုင်း အလိုအလျောက် ထွက်ပေါ်လာမည် ဖြစ်ပါသည်။
* Copy ခလုတ်ကို နှိပ်၍ မိမိအသုံးပြုလိုသော နေရာတွင် Paste လုပ်၍ အသုံးပြုနိုင်ပါသည်။
